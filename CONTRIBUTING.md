# Contributing to MD Files Connector

Thank you for your interest in contributing!

## Setup

```bash
git clone https://github.com/Maneesh-Relanto/md-connector
cd md-connector
pip install -r requirements-dev.txt
```

## Running locally

```bash
python md_connector.py .
```

## Running tests

```bash
pytest tests/ -v --cov=md_connector --cov-report=term-missing
```

## Submitting a PR

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Open a pull request

Please make sure your PR includes updates to documentation if needed.
